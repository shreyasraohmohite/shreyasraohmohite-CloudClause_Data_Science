# CodeClause_Loan_Prediction

## Loan Prediction Using Python

The project is based on Loan Prediction to approve the loan or not

Python - IDE Used - Jupyter Notebook(Anaconda)
Libraries Used
1.pandas
2.Sklearn
3.tkinter
4.joblib


It Uses Various Algorithms of Machine Learning

Algorithms Used
1. Logistic Regression
2. SVM(Support Vector Machine)
3. Decision Tree Classifier
4. Random Forest Classifier
5. Gradient Boosting Classifier
6. Hyperparameter Tuning

Data_Sets Used - https://www.kaggle.com/datasets/altruistdelhite04/loan-prediction-problem-dataset
Rename the data sets to loan_prediction.csv

All the files should be at same directory else it will not work.
